# 💰 Expense Tracker

A modern and responsive **Expense Tracker** built using **HTML, CSS, and JavaScript**. This application helps users manage their daily expenses by adding, searching, filtering, and deleting transactions while automatically calculating the total amount spent. All data is stored locally using the browser's **Local Storage**, ensuring persistence even after refreshing the page.

## ✨ Features

- ➕ Add new expenses
- 🗑️ Delete existing expenses
- 💰 Automatic total expense calculation
- 🔍 Search expenses by title
- 🗂️ Filter expenses by category
- 💾 Local Storage support (data persists after refresh)
- 📱 Fully responsive design
- 🎨 Clean and modern user interface
